CSS Variables Reminder

22 Feb, 2024

CSS variables are defined with two dashes:

:root {
  --primary-color: #3182ce;
}

And used with the var() function:

.button {
  background-color: var(--primary-color);
}

This enables easy theming and maintenance!

Mobile Development Tips

15 Feb, 2024

When developing for mobile, always test on real devices. Simulators can miss many real-world issues related to performance, touch sensitivity, and network behavior.

Remember that for iOS, the safe area insets are crucial for modern devices with notches!

Git Branching Strategy

30 Jan, 2024

For new projects, consider a simplified Git workflow:

  1. main - Production code
  2. develop - Integration branch
  3. Feature branches - Named with feature/name-of-feature

This keeps things clean while still providing isolation for new features.
